# Crenova Project

This is a static HTML website for a food restaurant/cafe.

## Structure

- `index.html` - Main homepage
- Various page HTML files (about-us.html, menu.html, etc.)
- `assets/` - Static assets
  - `css/` - Stylesheets
  - `js/` - JavaScript files
  - `images/` - Images
  - `fonts/` - Fonts

## Technologies

- HTML5
- CSS3 (Bootstrap, custom styles)
- JavaScript (jQuery, plugins)
